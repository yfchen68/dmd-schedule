## 系統簡介

本系統供數位媒體設計學系教師使用，主要用途有三：

1. **課程預排**：在正式排課前，預先規劃各課程的上課時段、教室與授課教師
2. **現況觀察**：以視覺化課表即時瀏覽本學年度所有課程的配置狀況
3. **歷年分析**：保存各學年度排課紀錄，可回顧過往課程開設情形與老師授課歷史

---

## 使用權限說明

使用 **任何 Google 帳號**登入後即可瀏覽與編輯所有課程，不需要額外申請或設定。

每次新增、編輯、刪除操作都會自動記錄操作者的 Google email，可在「歷史紀錄」中查閱。

## 主要功能

### 課表視圖

- 以週課表（週一至週六 × 第 1–12 節＋中午）呈現所有課程
- 每張課程卡片顯示：課號、課名、授課老師、教室、節次、人數、課程狀態
- 可依**學年度、老師、教室、年級、學期**篩選，快速聚焦特定條件
- 支援全文搜尋（課號、課名、老師、教室）

### 清單視圖

- 以表格列出全部課程，欄位包含：課號、班別、授課老師、學年度、適用年級、年/學期、每週節次、教室、上課時間、人數、狀態
- 手機瀏覽時自動切換為卡片式排版

### 新增與編輯課程

每門課程可設定以下資訊：

| 欄位 | 說明 |
|------|------|
| 課號 | 例：DMD2301，支援自動補全 |
| 課程名稱 | 支援自動補全（依過去紀錄） |
| 班別 | 未分班 / A–F 班 |
| 授課老師 | 可填多位，以逗號分隔 |
| 學年度 | 例：113、114 |
| 主要年級 | 一年級 / 二年級 / 三年級 / 四年級 / 研究所 |
| 適用年級 | 可多選，供跨年級選修課程使用 |
| 學期 | 上學期 / 下學期 |
| 課程性質 | 必修 / 選修 |
| 學分數 | 1–6 學分 |
| 人數上限 / 目前人數 | 用於顯示修課狀況 |
| 狀態 | 自動判定（開課中 / 準備中 / 已結束）或手動設為「停開」 |
| 上課時間 | 點選互動式時段格，或直接輸入，例：`一3,一4,一5` |
| 教室 | 例：D302，支援自動補全 |
| 備註 | 自由填寫 |

### 拖曳搬移時段

- 單日連續節次的課程可直接在課表上**拖曳**到其他時段
- 放下前會顯示確認視窗，防止誤操作

### 衝堂偵測

系統自動偵測教室衝堂，並以紅色框線與橫幅提示：

- **教室衝堂**：同一間教室在相同時段被重複使用

### 歷史紀錄與版本還原

- 每次新增、編輯、刪除都會記錄操作時間與操作者 email
- 可在「歷史紀錄」中瀏覽完整修改記錄
- 支援**一鍵還原**到任意歷史版本，還原前會顯示確認視窗

### 通知機制

以下操作會自動寄送 email 通知給系統管理者：

- 刪除課程
- 還原歷史版本
- 將課程狀態設為「停開」

### 匯出功能

- 匯出為 **CSV 檔案**（Excel 可直接開啟），包含所有欄位
- 支援**列印 / 存成 PDF**，列印版本僅顯示課表，並附上篩選條件標題

## 使用說明

### 瀏覽課表

打開系統網址，用 Google 帳號登入後即可使用所有功能。

### 篩選特定老師的課表

在上方工具列的「老師」下拉選單選擇姓名，課表即時更新，可搭配「學年度」、「學期」進一步縮小範圍。

### 列印單一老師的課表

1. 在「老師」篩選選單選擇該老師
2. 點「列印 / PDF」
3. 列印版本自動加上「授課老師 / 學年度 / 學期」標題

### 新增課程

1. 點右上角「＋ 新增課程」
2. 填寫課號、課名、老師等資訊
3. 在下方互動式時段格點選上課節次
4. 點「儲存」

### 編輯課程

直接點擊課表上的課程卡片，或在清單視圖點擊任一列，即可開啟編輯視窗。

### 查看歷史紀錄

點右上角「歷史紀錄」，可瀏覽所有修改記錄，並可點「還原」回到任意歷史版本。

## 資料儲存

所有資料儲存於 Google Sheets，分為兩個分頁：

- `courses`：課程資料，每列為一門課程班次
- `log`：操作記錄，每列含時間戳、操作者 email、動作類型與完整快照（供還原使用）

---

## 注意事項

- **停開**課程仍保留在課表中，不會自動刪除，可透過篩選隱藏
- **拖曳**功能僅適用於單日連續節次的課程，跨日或非連續節次請用編輯視窗修改
- 若同時有多人編輯，以最後儲存的版本為準；如有疑慮可透過歷史紀錄核對
- 每次操作均記錄操作者 email，請使用個人帳號登入
